# Внести вклад

## Руководство для контрибьюторов

Мы приветствуем любой вклад! Простой рабочий процесс для участников:

1. Создайте форк репозитория на GitHub.
2. Создайте ветку с понятным названием, например `feature/add-README` или `fix/whitelist-load`.
3. Внесите изменения и добавьте короткое сообщение о коммите.
4. Запустите локальные проверки, если они доступны.
5. Загрузите ветку в свой форк и создайте Pull Request в основной репозиторий.

<sup>Советы: опишите изменения и шаги тестирования в PR. Для крупных изменений разделите их на более мелкие коммиты.</sup>
