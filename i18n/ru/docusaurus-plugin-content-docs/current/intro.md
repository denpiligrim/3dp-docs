# Введение

Генератор входящих подключений для **3x-ui**.

## О проекте

### Обзор

Утилита для автоматического создания входящих соединений для панели [3x-ui](https://github.com/MHSanaei/3x-ui), создания единого URL-адреса подписки и настройки перенаправления трафика с промежуточного сервера на главный сервер.

### Основные цели

Основная цель утилиты — сделать ваш трафик неоднородным. Бот генерирует 10 записей подключений с настроенным интервалом и различными параметрами:

- Протоколы: `vless`, `vmess`, `shadowsocks`, `trojan`;
- Порты: `443`, `8443` (фиксированные) и случайные порты из диапазона `10000-60000`;
- Транспорт: `tcp`, `websocket`, `grpc`, `xhttp`;
- Значения SNI берутся из белого списка доменов (`whitelist`); вы можете использовать свой собственный список.

Все сгенерированные соединения объединяются в одну подписку со статическим URL. Бот работает с панелью `3x-ui`, используя ее публичный API, и не изменяет непосредственно внутреннюю часть панели.

Вторичной целью является стабильность соединения: клиент получает 10 альтернативных вариантов соединения и может выбрать любой из них.

Кроме того, бот можно использовать в каскадной конфигурации. Служба переадресации автоматически настроит подписку и перенаправление трафика на главный сервер.

## Рекомендации

- Используйте HTTPS для подписки (домен + SSL-сертификат).
- Установите интервал генерации ≥ 10 минут; для стабильности рекомендуется один раз в день (1440 минут).
- Настройте клиент на более частую проверку обновлений (например, каждый час), чтобы оставаться синхронизированным с сервером.
