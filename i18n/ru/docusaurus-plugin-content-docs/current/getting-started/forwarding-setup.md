# Настройка переадресации

Служба переадресации позволяет перенаправлять трафик на основной сервер и используется в каскадной схеме VPN. Инструмент перенаправляет все пакеты `TCP` и `UDP` на следующих портах: `443`, `8443`, `10000:60000`.

:::warning
Служба переадресации работает на промежуточном сервере.

:::

## Установка службы переадресации

В разделе `Перенаправление` добавьте новый сервер, введя название, IP адрес и пароль от SSH. Далее нажмите `Установить`. После установки перейдите в раздел `Подписки` и выберите ваш relay сервер, чтобы скопировать ссылку на подписку.
