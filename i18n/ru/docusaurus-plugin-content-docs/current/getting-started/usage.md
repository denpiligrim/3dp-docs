# Использование

## Показать URL подписки

Команда для вывода текущего порта панели управления:

```bash
docker port 3dp-frontend
```

<small>Краткое описание: выводит порт панели управления 3DP-MANAGER.</small>

Скопируйте порт из перовой строки 80/tcp -> 0.0.0.0:`PORT`.Далее откройте вашу панель по шаблону: `http(s)://IP_OR_DOMAIN:PORT`

## Сбор доменов из нескольких подписок

Эта утилита извлекает домены из мультиподписок и генерирует `белый список` для генератора.

```bash
node get_domains.js
```

<small>Краткое описание: добавьте URL-адрес с несколькими подписками в скрипт и запустите команду — она выведет список доменов. Требуется `Node.js`.</small>

## Использование кастомнго белого списка

1. Подготовьте файл в формате `whitelist.txt`.
```text title="my_whitelist.txt"
ya.ru
vk.com
miro.com
cloudflare.com
google.com
```
2. В разделе Домены нажмите `Добавить из файла` и выберите ваш файл `whitelist.txt`.

## Ручное выполнение генерации

В разделе `Настройки` нажмите `Сгенерировать сейчас`. Это вызовет генерацию новых инбаундов для всех подписок, не влияя на интервал ротации.
